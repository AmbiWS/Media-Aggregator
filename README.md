# Media Aggregator

App, developed in 'junior to middle course' of Mikhail Valuyskiy, Mobile TeamLead in Beeline.  
Simple movie aggregator.  
Tools and etc.: MVVM, RxJava, Room, Retrofit, LiveData, Dagger.
